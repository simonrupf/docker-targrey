# taRgrey

S25R + tarpitting + greylisting - a tarpit + greylisting policy server.

http://k2net.hakuba.jp/targrey/index.en.html

## Usage

```shell
make run
```

## Stop

```shell
make clean
```

## Build

```shell
make build
```

## Build & run

```shell
make
```

## Debug image contents

```shell
make debug
```
